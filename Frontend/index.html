Ce projet présente le résultat de nos hypothèses concernant le choix de le filière d'études, ASTRE/IPS, par les étudiants de l'Ensim. Vous pourrez modifier les poids de chaque hypothèse en bougeant les curseurs pour chaque question avec 0 signifie que l'hypothèse n'aura aucune implication sur le résultat et 10 signifie que l'hypothèse est la plus importante. La page proportions presente la prédiction sur l'orientation de la étudiants de classe vers ASTRE ou IPS. La page détails affiche chaque étudiant avec son score. Pour réinitialiser les poids, rechargez la page.
